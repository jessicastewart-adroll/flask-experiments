How do we hyperlink such that our own Flask page is returned? We use the Flask function url_for(). Because this function is considered a 'Python experssion' it must be wrapped in double curly braces in order to be compiled as Python. Try clicking on this 'url for' the css page.

  • The route decorator takes the 'rule' argument which is simply the URL that will display
  • The url_for function takes the 'endpoint' argument which is the name of the function that will return the page content.
      • view function/endpoint = function that returns page content
      route/rule = display string appended to URL
